Description

The HYST keyword activates the hysteresis option, the keyword should be avoided and the hysteresis option should be enabled by the HYSER parameter on the SATOTPS keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
